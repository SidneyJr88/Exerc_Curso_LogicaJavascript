Lógica de Programação Javascript: Exerc 04

  1. Jogador 1 digita um numero;
  2. Jogador 2 digita um numero;
  3. Numeros iguais mostrar mensagem : "Empate!";
  4. Sortea numero entre 0 e 1;
  5. Numero sorteado igual a 0, ganha quem escolheu o menor numero;
  6. Número sorteado igual a 1, ganha quem escolheu o maior número.
