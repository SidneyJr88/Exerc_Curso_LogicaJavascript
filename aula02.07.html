Lógica de Programação Javascript: Aula 2.7

Condicionais

  1. Usuário digita duas notas
  2. Média de corte 5
  3. Mostra a mensagem de aprovação ou não

Código utilizado

			var nota1 = parseFloat(prompt("Digite nota 1"));
			var nota2 = parseFloat(prompt("Digite nota 2"));
			var media = (nota1+nota2)/2;

			if (media>5) {
				alert("Aprovado");
			}
			else{
				alert("Reprovado");
			}
