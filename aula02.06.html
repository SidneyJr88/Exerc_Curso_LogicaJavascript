Pedir para o usuário digitar dois números e mostrar um boolean True(verdadeiro) ou False (falso) indicando se a média for maior que 5


			var num1 = parseFloat(prompt("Digite um numero"));
			var num2 = parseFloat(prompt("Digite o segundo número"));
			
			var media = (num1+num2)/2;
	
			alert(media>5);
